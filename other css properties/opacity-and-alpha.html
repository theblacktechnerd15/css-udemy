Opacity and Alpha

Alpha: The alpha channel in RGBA governs the transparency of the color. It is a value from 0-1.
0 - Transparent
1 - Opaque

Opacity: Governs the transparency for an entire element
0 - Transparent
1 - Opaque

For hexadecimals, an extra 2 number values are added for transparency scaling from 00 (Transparent) to FF (Opaque).
e.g: #00fca00d
