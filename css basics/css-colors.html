CSS Colors

There are 3 main types of color values in html.

1. Named Colors

Using color names. This method is limited

2. Hexadecimals

Uses a mixture of letters and numbers ranging from 0-F. Every 2 Number Values represents RGB.
e.g: #000000 and #FFFFFF

3. RGB

RGB (Red, Green, Blue) has 3 color channels each ranging from 0-255. 0 being the lightest and 255 being the darkest.
e.g: color: rgb(255,255,255) BLACK
