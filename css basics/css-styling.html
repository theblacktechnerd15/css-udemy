CSS Styling Methods

    There are 3 ways to style elements using CSS

  • Inline Styles: Styling an element using the style attribute. (Not Reccomended)
  • The < style > Element: inserting the <style> element in the <head> tag. (Not Reccomended)
  • External style sheet: <link> to the css file in your head tag (html)
    Use the link tag in the head section
