Text Properties

text-align

Aligns the text on the page
values: left, right, center, justify

font-weight

Sets the font weight (or boldness) of the font. The weights available depend on the font-family that you are using.
values: normal; bold; lighter; bolder; 100-800
Note:Headers by default have bolded fonts.

text-decoration

The text-decoration is a shorthand CSS property that sets the appearance of decorative lines on text.
Shorthand for:
text-decoration-line
text-decoration-color
text-decoration-style
text-decoration-thickness
values: solid, dotted, double, dashed, underline, none.

line-height

Changes the spacing between each line.
values: normal (1.2px), width/height , a percentage , a number
Unitless value: line-height: 3.5; (Meaning 3.5x the font-size)

letter-spacing

Sets horizontal spacing values between text characters
values: normal, width/height
