CSS SELECTORS

Universal Selector (*)

Selects everything on the page
Syntax:
*{
property:value;
}

Element Selector

Selects an Element
Syntax:
element{
property:value;
}

Selector List (,)

Selects seperate elements
Syntax:
element, element{
property:value;
}

The ID Selectors(#)

Selects only ids
Syntax:
#id{
property:value;
}

The class selector (.)

Selects classes only
Syntax:
.job-section{
property:value;
}

The descendant selector

Selects the child element of the parent
Syntax:
li(p) a(c){
property:value;
}

Adjacent Selector (+)

Selects the target element right after the primary/former element
Syntax:
(former element)h1 + (target element)p{
property:value;
}

Direct Child Selector (>)

Selects the direct child element of the parent (Not nested only direct)
Syntax:
div>li{
property:value;
}

Attributes Selector

Selects attributes from an element
Syntax:
element[attribute="value"]{
property:value;
}
