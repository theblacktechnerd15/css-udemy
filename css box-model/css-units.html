Common CSS Units

Relative:

em / rem:
Ems are relative units. With font-size, 1 em equals the font-size of the parent. 2 ems is twice the font-size of the parent , etc.

With other properties, 1 em is equal to the computed font-size of the element itself.
(ems are commonly used with margin/paddings)

Percentages %:
Always relative to some other value. Somtimes its a value from the element itself.
width: 50% - half the width of the parent.
line-height: 50% - half the font size of the element itself.

% commonly used with width/height
