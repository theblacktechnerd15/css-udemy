Box Model and Height

Border / Border Radius

border-width:
Controls the thickness of the border. (Use px measurement)

border-color:
Controls the color of the border.

border-style:
Controls the border line style .
e,g: dashed, dotted, solid etc.

box-sizing :
Sets how the total width and height are calculated.

border (shorthand property) :
border{
thickness, width, style, color;
}

border-radius :
adds rounded corners. (reccommended to use pixels over percentages)
