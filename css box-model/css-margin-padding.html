Padding/Margin (Box-Model)

Padding is the space between the content box and border (Displays Green on Chrome Dev)
Margin is the space outside of the border (Displays Orange on Chrome Dev Tools)

Individual Properties

padding-left
padding-right
padding-bottom
padding-top

Shorthand Property

padding: Sets all 4 sides at once

Padding Shorthand

1.Apply padding value to all 4 sides.
padding: 10px

2. Vertical / Horizontal
padding: 5px 10px;

3. top/ horizontal /bottom
padding: 1px 2px 2px;

4.top/right/bottom/left
padding: 5px 1px 0 2px;

Margin: Always common to set body page margin to 0

Note: All above relates to the margin property as well
